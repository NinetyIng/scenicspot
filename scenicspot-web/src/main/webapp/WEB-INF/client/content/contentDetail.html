检邮携手 开通都匀市"预防职务犯罪邮路"
2017-04-03 03:25:04

为将检察机关惩治和预防职务犯罪职能与邮政服务社会群众工作有机结合，创新“检察+邮政”的工作模式，日前，都匀市人民检察院、中国邮政集团公司都匀市分公司联合举行“预防职务犯罪邮路”活动启动会，出台两家关于联合开展“预防职务犯罪邮路”活动实施工作方案，聘请市邮政分公司19名工作人员为职务犯罪预防邮路志愿者。

出席启动会的中国邮政集团黔南州分公司副总经理岳文秀指出，都匀市分公司下属各支局、各条邮路及网点全覆盖都匀市各乡镇办事处，市分公司有信心、有能力、有条件与都匀市人民检察院一起将预防邮路打造成清风路、和谐路、文明路。
